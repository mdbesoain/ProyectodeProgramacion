Tutorial FotoChop

Introducción

El presente tutorial tiene por finalidad guiar al usuario en el uso del software, en el cual podrás generar múltiples funciones a partir de las opciones disponibles, dentro de las cuales encontraras opciones de edición disponibles en el campo de “EDITOR”, además podrás imprimir, guardar nuevas imágenes, visualizar, crear tus propios filtros y cargarlos al software, entre muchos otros. Esperamos que el siguiente tutorial pueda aclarar y ser de ayuda para que puedas obtener el mayor provecho a esta herramienta.

2)   Funcionalidades de la barra de herramientas

La barra de menú se compone de múltiples opciones las cuales hemos agrupado en módulos, a continuación explicaremos el Módulo 1:

Módulo 1 [Figura 2.1]

a)    Archivo

b)   Edición

c)    Ver

d)   Ayuda

e)    Herramientas

 

         

                                                                       [Figura 2.1: MÓDULO 1]

 

a)    Archivo [Figura 2.1.1]: Ítem “Archivo” permite realizar las operaciones de crear NUEVO ventana de edición, ABRIR imagen, GUARDAR avances, GUARDAR COMO permite dar una ubicación a la imagen creada, permite además asignarle el formato JPG, BMP o PNG, CONFIGURACIÓN DE PAGINA modificar el formato de la página para cuando desee imprimir el nuevo proyecto, ya sea en tamaño Carta, A4, Legal, etc., permite modificar márgenes y orientación. Finalmente IMPRIMIR permite prepara la página para ser impresa.

3)   Funcionalidades de Editor

Dentro de las funcionalidades que Editor tiene a disposición, podemos generar 7 módulos, estos módulos se enuncian y se describen a continuación:

Módulo 1 [Figura 3.1]

a)    Filtros pre-establecidos

b)   Filtros manuales

 

                                   [Figura 3.1.1: FILTROS pre-establecidos y manuales]

a)    Filtros pre-establecidos [Figura 3.1.2] permite al usuario seleccionar cualquier tipo de filtros disponibles por el Scroll, dentro de los cuales encontramos los filtros de: Blanco y Negro, Borde Frontera, Brillo, Escala de Grises, FloodFill, Invertir X, Invertir Y, Negativo, Polarizado, Priorizar Rojo, Priorizar Verde, Priorizar Azul, RGB to GBR, RGB to RBG, Sepia.

 

 

                                         [Figura 3.1.2: FILTRO Cuarteado]

 

b)   Filtros manuales [Figura 3.1.3] permite crear nuevas combinaciones de colores con los controles manuales de RGB, acerca el slider hacia la derecha y obtendrás colores con tonalidades más blancas, acerca el slider hacia la izquierda y obtendrás colores con tonalidades más negras. Puedes interactuar con los tres colores primarios: Rojo (R), Verde (G) y Azul (B).

 

[Figura 3.1.3: FILTROS Manuales]
